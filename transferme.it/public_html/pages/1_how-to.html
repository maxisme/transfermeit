Send a file?

There are three different ways to simply and efficiently send your file!

Drag a file from Finder onto the menu bar icon.

Or

Right Click a file in Finder and choose Send File (or: Services > Send File).

Or

Click on the menu bar icon then click Send File and choose your file.

After picking your file...

Enter your friends code and hit send.

Create a code?

By default transfer me it will create a 10 minute code to identify you. This means a friend can only identify (and send a file to) you within 10 minutes. The server will also only store your encrypted file on the server for a maximum of the set minutes (files are immediately deleted once your friend has downloaded the file). Depending on your credit you can choose up to 1 hour and whether your code is permanent and unique.
